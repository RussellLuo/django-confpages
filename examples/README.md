# Examples

Some example Apps based on `Django-ConfPages`.


## Dependencies

1. Install `Django-ConfPages`.

    ```
    $ pip install Django-ConfPages
    ```

2. Download examples

    ```
    $ cd /tmp
    $ git clone https://github.com/RussellLuo/django-confpages.git
    $ cd django-confpages/examples
    ```
